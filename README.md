## Disintangled Analytics
This model combines my DSL language with Get Statistics in order to create an entirely new program based on its weights.

### Maisette DSL

To perform quiquandquo:

~~~ruby
quiquandquoi("Anatta",               "travailles avec il velo a l'ecole",   "en le heure de 6:00 A.M.",
                 cette("velo", "rouge"), maisette("l'ecole", "vert"),           sinon("route", "rouge", "vert"))
~~~

To perform get statistics:

~~~ruby
2.times do
  get_statistics(:anatta, a,
                 :pierre, b,
                 :abe,    c)

                 dynamic_reward_allocation
end
~~~
